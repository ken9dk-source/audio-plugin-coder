# Creative Brief: SID Trance Machine (C-MOS)

## Hook
**"The ghost of Commodore 64 reborn as a 2006 uplifting trance weapon."**
Three SID-inspired oscillators. One merciless filter. Pure trance energy.

## Description
SID Trance Machine (codename: C-MOS) is a hybrid synthesizer that fuses the
gritty, aliased character of the MOS Technology 6581/8580 SID chip with the
polyphonic power and stereo width demanded by modern trance music production.

Where classic C64 emulators stay faithful to 8-bit limitations, C-MOS breaks
free: it adds a 16-voice polyphonic engine, a stereo unison stack, and a
full effects chain (chorus, delay, reverb) tuned for the emotional arc of
uplifting and psytrance. The result sits between Sylenth1's warmth, Diva's
analog drift, and the raw digital edge that no VST has ever fully captured.

## Sonic Identity
- **Leads:** Gritty SID saw leads with TRANCE DRIFT detuning and analog glow saturation
- **Plucks:** Short-decay triangle+pulse combos through a snapping 24dB LP filter
- **Pads:** Slow-attack unison stacks with chorused stereo width
- **Basses:** Resonant SID filter sweeps with Juno-like warmth and digital age bit character

## Unique Features (USPs)
| Feature | Description |
|---|---|
| **TRANCE DRIFT** | Subtle random pitch micro-drift per voice, emulating chip oscillator instability |
| **DIGITAL AGE** | Bit-reduction / sample-rate crush blend for authentic SID aliasing character |
| **SID WIDTH** | Stereo unison spread engine (up to 7 voices per note) |
| **ANALOG GLOW** | Soft saturation stage inspired by Juno chorus + SID output stage warmth |

## Mode System
- **Classic SID** — Raw chip behavior, mono/limited poly, aliasing allowed
- **Modern SID** — Clean polyphony, bandlimited oscillators, reduced aliasing
- **Trance SID** — Full stereo unison, TRANCE DRIFT, FX enabled, 16 voices

## Visual Identity
Dark navy blue chassis. Cyan and electric blue neon controls. Green resonance
ring. Waveform oscilloscope display at top. "SID TRANCE MACHINE / C-MOS /
MOS Chip Inside" branding. Future-retro — not Commodore beige, but a C64
reimagined as a rack unit from the year 3000.

## Reference Synths
- reFX QuadraSID (SID architecture)
- Plogue chipsynth C64 (SID authenticity)
- Sylenth1 (trance pad/pluck workflow)
- Access Virus TI (filter character + build quality feel)
- u-he Diva (analog drift modeling approach)
